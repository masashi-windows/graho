Graho
=====

**This iOS App is in development.**
**Please waite a second.**

The Graho is an iOS App. It is a support tool for making weekly reports.

**Grahoはまだ開発中です。**

Grahoは週報を作成するiOSアプリです。  
1週間の業務内容、体調、勤務時間をグラフで表示し、SNSやメールなどで通知するアプリです。

来年の頭にはストアにアップしたい。
